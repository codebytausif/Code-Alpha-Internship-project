# 🖼️ Image Gallery

A responsive Image Gallery built using HTML, CSS, and JavaScript as part of my Frontend Development Internship at CodeAlpha.

## 📌 Overview

This project allows users to browse and filter images through different categories. It is designed with a clean user interface and responsive layout to provide a smooth viewing experience across devices.

## ✨ Features

- Responsive Design
- Image Categories Filtering
- Modern User Interface
- Smooth User Experience
- Mobile Friendly Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

Through this project, I improved my understanding of:

- Responsive Web Design
- CSS Grid & Flexbox
- DOM Manipulation
- JavaScript Event Handling
- UI Design Principles
